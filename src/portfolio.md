## 关于麦芽糖bot

麦芽糖bot是一个功能丰富的QQ机器人，我们致力于为用户提供稳定、易用的机器人服务。

### 主要特性

- **娱乐功能**: 提供多种娱乐游戏和互动功能
- **实用工具**: 天气查询、翻译、等实用工具
- **完全免费**: 完全免费，用爱发电

### 技术栈

- **后端**: Node.js + TypeScript + Python 
- **框架**:  Koishi + GsCore + Karin + NapCat
- **数据库**: SQLite / MySQL
