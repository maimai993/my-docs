# 图像处理

麦芽糖bot提供丰富的图像处理功能，让您的图片更加有趣和实用。

## 功能列表

### 🎭 表情包制作
- **[表情包生成](./表情包生成.md)** - 自定义表情包生成
- **[啤酒烧烤表情包生成](./啤酒烧烤表情包生成.md)** - 啤酒烧烤风格表情包
- **[随机表情包](./随机表情包.md)** - 随机获取表情包
- **[VV表情包搜索](./VV表情包搜索.md)** - 搜索VV表情包

### 🖼️ 图片处理
- **[GIF 图片处理](./GIF图片处理.md)** - GIF动图处理功能
- **[随机图片](./随机图片.md)** - 随机获取图片
- **[拼接图片](./拼接图片.md)** - 多张图片拼接
- **[幻影坦克生成](./幻影坦克生成.md)** - 幻影坦克效果生成
- **[原图坦克](./原图坦克.md)** - 原图坦克效果
- **[相机镜框](./相机镜框.md)** - 添加相机镜框效果
- **[像素化](./像素化.md)** - 图片像素化处理
- **[光棱生成](./光棱生成.md)** - 光棱效果生成
- **[emoji合成](./emoji合成.md)** - Emoji表情合成

### 🏆 创意生成
- **[生成奖状](./生成奖状.md)** - 自定义奖状生成
- **[二维码](./二维码.md)** - 二维码生成
- **[Markdown 渲染](./Markdown渲染.md)** - Markdown转图片

## 使用说明

1. **命令格式**：`[命令] [参数]`
2. **图片上传**：部分功能需要上传图片作为输入
3. **参数说明**：仔细查看每个功能的参数要求
4. **获取帮助**：使用 `帮助` 查看详细使用说明

选择您需要的图像处理功能开始创作吧！
